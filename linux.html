What is Linux

"Just like Windows, iOS, and Mac OS, Linux is an operating system. In fact, one of the most popular platforms on the planet, Android, is powered by the Linux operating system. An operating system is software that manages all of the hardware resources associated with your desktop or laptop. To put it simply, the operating system manages the communication between your software and your hardware. Without the operating system (OS), the software wouldn’t function." according to linux.com

What is the terminal

"A terminal is simply a text-based interface to the computer. In a terminal, you can type commands, manipulate files, execute programs, and open documents. When working in a terminal, the current directory is called your working directory. A terminal will usually start in the top-level directory of your account. "according to cs.colby.edu

In the table below, you will see a list of common commands in the linux terminal
Command What it does
cd "directory name" changes directory into the selected directory
cd .. Goes back a directory
ls lists the contents in the current directory
pwd Prints the working directory
whoami tells you what user you are signed in as
mkdir "directory name" Makes a directory
rm "file name" removes a file from the current directory
rmdir "directory name" removes a directory from the current directory
sudo -i signs in as a super user
