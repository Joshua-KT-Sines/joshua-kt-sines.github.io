Basic Overview

While most ETHICAL hackers ussaly build their own tools with Python or Bash, there are some common tools that are universal. Some of the most popular hacking tools are the following: NMAP, Exploit DB, Nessus Scan, and Burpsuite. NMAP, a tool used to scan for open network ports, and server info is commonly used by people all over the tech industry. Exploit DB is a database of thousands of known exploits and vulnerabilities. Nessus Scan is a web base tool that lets you scan for vulnerabilities for software like vsftp, and apache. Burpsuite is a tools used for SQL injectoins, this tools paired with a brute force tool like hydra, can make password cracking very easy.
